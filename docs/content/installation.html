Installation

PullApprove is installed as a GitHub App. You can either choose specific repos within your organization, or install it for all repos in an organization.

We will only report a status check to GitHub if you have a .pullapprove.yml file using version: 3 in the repo. Versions 1 and 2 were built using the original OAuth integration system, and can still be used side-by-side with version 3.
