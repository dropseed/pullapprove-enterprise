Config validator

{% markdown %} To test and validate a `.pullapprove.yml` file, [use the command `/pullapprove-test` in a PR review comment](/commands/). Your config will be tested against the live pull request (without sending any changes or statuses) so that you can test conditions and other settings with real data. The results of the test will be sent to the same PR in a comment.
